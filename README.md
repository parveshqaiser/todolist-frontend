🚀 SprintDo – Frontend

SprintDo is a modern and minimal Todo List Application designed to help users manage their daily tasks efficiently. It allows users to add tasks, track progress, monitor completion rates, and view recently created tasks — all in a clean and responsive interface.

<br/>

📌 Features

✅ Add daily tasks

🗂 Track task status (Completed / Pending)

📊 View task completion rate

🕒 Track most recently created tasks

🔐 Persistent authentication with automatic token refresh

📱 Fully responsive design

<br/>

🛠 Tech Stack

React – Frontend library for building UI

TailwindCSS – Utility-first CSS framework for styling

Jotai – Lightweight state management

Bundler - Vite

Axios – HTTP client with interceptor setup for automatic token refresh

<br/>

⚙️ Installation & Setup

Clone the repository: git clone 

Navigate into the project folder: cd sprintdo-frontend

Install dependencies: npm install

Start development server:npm run dev 🌟
